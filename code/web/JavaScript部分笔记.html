JavaScript 数据类型



JavaScript 中有五种可包含值的数据类型：


  • 字符串（string）
  • 数字（number）
  • 布尔（boolean）
  • 对象（object）
  • 函数（function）

有三种对象类型：


  • 对象（Object）
  • 日期（Date）
  • 数组（Array）

同时有两种不能包含值的数据类型：


  • null
  • undefined

javascript类型转换

JavaScript 变量能够被转换为新变量以及另一种数据类型：

  • 通过使用 JavaScript 函数
  • 通过 JavaScript 本身自动转换
